이 영상은 Cloudflare의 혁신적인 AI 코드 리뷰 시스템을 분석하며, 어떻게 S 티어 토크노믹스를 달성했는지 상세히 파헤칩니다. 개발자는 직접 구축한 에이전트 하네스와 전문화된 에이전트들을 통해 비용 효율성을 극대화하고, 이를 통해 엔지니어의 시간을 절약하며 막대한 가치를 창출합니다.

1.  **S 티어 토크노믹스 달성**
    *   Cloudflare는 병합 요청당 1달러라는 혁신적인 비용으로 코드 검토를 수행하며, 5,000개 코드베이스 규모에서 막대한 토큰 차익 거래를 실현했습니다.
    *   이는 최첨단, 워크호스, 경량 모델 스택과 공격적인 프롬프트 캐싱, 공유 컨텍스트 파일, 도메인별 diff 패치 스코핑을 통해 가능했습니다.

2.  **하네스 엔지니어링 및 에이전트 전문화**
    *   폐쇄된 솔루션을 임대하는 대신 OpenCode 기반의 맞춤형 에이전트 하네스를 직접 구축하고 프로그래밍 가능한 SDK 액세스를 활용했습니다.
    *   보안, 성능, 코드 품질 등 7개의 전문화된 리뷰어 에이전트가 단일 범용 에이전트보다 훨씬 뛰어난 성능을 발휘합니다.

3.  **컨텍스트 및 프롬프트 엔지니어링 최적화**
    *   공유 컨텍스트 파일과 도메인별 diff 패치를 사용하여 전체 병합 요청을 반복적으로 처리하는 7배의 토큰 낭비를 방지했습니다.
    *   각 에이전트가 무엇을 보고 무엇을 무시해야 하는지 명확히 지시하는 엄격하게 범위가 지정된 프롬프트는 불필요한 구문 검토를 줄입니다.

4.  **모델 유연성 및 시스템 복원력 확보**
    *   작업에 맞는 적절한 모델을 배포하기 위해 공급업체별 대체 기능을 갖춘 계층화된 모델 스택을 활용합니다.
    *   타임아웃, 재시도 예산, 단계 완료 트리거, 오류 분류 및 모델 대체 기능을 통해 진정한 아웃루프 안정성을 보장합니다.
