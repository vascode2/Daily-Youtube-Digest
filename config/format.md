# 요약 출력 포맷

## 채널 헤더
각 채널 섹션은 h3 헤딩 + 📺 + **채널명(핸들 아님)** + **YouTube 채널 링크**:

```markdown
### 📺 [채널명](https://www.youtube.com/@CHANNEL_HANDLE)
```

예시:
- `### 📺 [AI Engineer](https://www.youtube.com/@aiDotEngineer)`
- `### 📺 [안될공학 - IT 테크 신기술](https://www.youtube.com/@unrealtech)`
- `### 📺 [데키랩](https://www.youtube.com/@dekilab)`

채널명과 핸들은 raw 데이터에서 사용. publish 단계에서 채널명이 빨간색으로 렌더링됨 (Notion `color: red`).

## 영상별 요약 포맷

각 영상 요약은 아래 형식을 정확히 따라야 합니다.

---

## [영상 제목](https://www.youtube.com/watch?v=VIDEO_ID)

**한 줄 인사이트**
💡 이 영상의 핵심 통찰을 한 문장으로. 본문에서 가장 중요한 한 가지 주장/숫자/판단을 골라 압축. (FAE 관점이나 "실무 적용" 같은 일반론을 강제하지 말 것 — 영상 자체의 내용에 충실)

**핵심 요약**
도입 1문장으로 영상의 전체 맥락(누구의 발언/인터뷰/데모인지, 무엇을 다루는지)을 잡아주고, 그 다음 **번호 + 굵은 소제목** 구조로 가장 중요한 내용만 압축합니다. 읽는 사람이 모든 영상을 다 보지 않아도 되도록, 세부사항보다 핵심 판단과 큰 흐름을 우선합니다.

각 번호 항목은:
- 굵은 소제목(주제 한 줄)
- 그 아래 1개의 sub-bullet만 사용
- sub-bullet 안에는 **구체적 인명/기업명/숫자/연도** 중 정말 중요한 것만 포함
- 핵심 요약 안에는 타임스탬프를 과하게 넣지 말고, 시간 이동은 아래 **주요 타임라인**에 맡깁니다.

분량 제한:
- 영상 1개당 번호 항목은 **최대 3개**
- 전체 sub-bullet은 **최대 4개**
- 각 bullet은 **1문장, 35~55자 수준**으로 압축

본문 요약 그 자체에 집중. takeaway 문장이나 "FAE 관점에서…" 같은 마무리는 **추가하지 말 것** — 영상이 말하지 않은 일반론을 덧붙이지 않습니다.

**주요 타임라인**
영상 전체 흐름을 대표하는 시간 링크 3개만 작성합니다.
- 형식: `- [[HH:MM](https://www.youtube.com/watch?v=VIDEO_ID&t=SECONDS)] 핵심 장면/주장 한 줄`
- 초반/중반/후반에서 각각 의미 있는 지점을 고릅니다.
- 긴 영상에서 초반 몇 분만 나열하지 않습니다.
- 자막 구간이 없으면 `[자막 기반 타임라인 없음]`을 1회 작성합니다.

영어 채널도 반드시 한국어로 작성합니다.

예시 골격:

```markdown
**핵심 요약**
실리콘밸리 대표 VC 마크 안드레센의 인터뷰 요약. 실수의 종류, 위대한 창업자 조건 등 정리.

1. **벤처 캐피털의 사고방식: '실수로부터 배우기'의 함정**
   - $10M 손실보다 Google 같은 기업을 놓치는 $100B 기회비용을 더 경계.

2. **위대한 창업자의 3가지 조건**
   - High IQ, 용기, 야망을 모두 갖춘 사람이 큰 회사를 만든다고 설명.

**주요 타임라인**
- [[03:00](...&t=180)] 실행의 실수보다 놓친 기회비용이 더 크다는 관점
- [[04:26](...&t=266)] 위대한 창업자를 판별하는 High IQ 기준
- [[05:30](...&t=330)] 단순 문제 해결을 넘어 자기 것을 구축하려는 야망
```

---

## 규칙
- 영상 제목은 **반드시 h2 헤딩 + YouTube 링크** 형식 (`## [제목](https://www.youtube.com/watch?v=VIDEO_ID)`)
- 조회수, 길이, 자막 유무, **업로드 날짜**는 출력에 포함하지 않음
- 날짜 정보는 페이지 제목에 이미 포함되어 있어 영상별 날짜는 불필요
- 섹션 순서: **한 줄 인사이트** → **핵심 요약** → **주요 타임라인**
- 핵심 요약은 도입 1문장 + **번호 매긴 소주제(굵은 소제목)** 최대 3개 + 각 소주제 아래 1개 sub-bullet 구조 사용
- sub-bullet에는 **구체적 인명·기업명·숫자·연도** 중 핵심 판단에 필요한 것만 인용 (모호한 일반 진술 금지)
- **주요 타임라인**은 3개 링크만 작성하고, 초반/중반/후반을 대표해야 함
- **핵심 요약 본문에 takeaway/FAE 관점/실무 적용 같은 일반론 덧붙이지 말 것** — 영상이 실제로 말한 내용만 정리
- 핵심 요약 본문에 발표자가 든 예시/사례/데모가 핵심 이해에 중요하면 1개 포함
- **핵심 요약 첫 문장을 영상 제목(또는 제목의 앞 부분)으로 시작하지 않음** — 제목은 이미 h2 헤딩에 있으므로 반복 금지
- **한 줄 인사이트는 모든 영상에 대해 서로 다르게 작성** — 같은 다이제스트 안에서 동일한 보일러플레이트 인사이트 금지
