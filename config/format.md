# 요약 출력 포맷

## 채널 헤더
각 채널 섹션은 h3 헤딩 + 📺 + **채널명(핸들 아님)** + **YouTube 채널 링크**:

```markdown
### 📺 [채널명](https://www.youtube.com/@CHANNEL_HANDLE)
```

예시:
- `### 📺 [AI Engineer](https://www.youtube.com/@aiDotEngineer)`
- `### 📺 [안될공학 - IT 테크 신기술](https://www.youtube.com/@unrealtech)`
- `### 📺 [데키랩](https://www.youtube.com/@dekilab)`

채널명과 핸들은 raw 데이터에서 사용. publish 단계에서 채널명이 빨간색으로 렌더링됨 (Notion `color: red`).

## 영상별 요약 포맷

각 영상 요약은 아래 형식을 정확히 따라야 합니다.

---

## [영상 제목](https://www.youtube.com/watch?v=VIDEO_ID)

**한 줄 인사이트**
💡 이 영상의 핵심 통찰을 한 문장으로. 본문에서 가장 중요한 한 가지 주장/숫자/판단을 골라 압축. (FAE 관점이나 "실무 적용" 같은 일반론을 강제하지 말 것 — 영상 자체의 내용에 충실)

**핵심 요약**
**2~3문장의 도입 단락**으로 영상의 이야기 흐름(누구의 발언/인터뷰/데모인지, 어떤 주장을 이끌어 가는지, 근거는 무엇인지)을 잡아주고, 그 다음 번호 + 굵은 소제목 구조로 내용을 정리합니다. 영상을 보지 않은 독자도 저자가 어떻게 주장을 쌓아가는지 이해할 수 있도록 맥락을 충분히 담습니다.

각 번호 항목은:
- 굵은 소제목(주제 한 줄)
- 그 아래 1~2개의 sub-bullet
- sub-bullet 안에는 자막에서 인용한 **구체적 인명/기업명/숫자/연도/제품명**을 포함해 자세히 설명
- 핵심 요약 안에는 타임스탬프를 과하게 넣지 않고, 시간 이동은 아래 **주요 타임라인**에 맡김니다.

분량 제한:
- 도입 단락: **2~3문장** (영상의 주제만 알리지 말고 이야기 흐름까지 드러내기)
- 영상 1개당 번호 항목은 **3~5개** (자막이 롤분하면 4~5개 권장, 3개로 인위적으로 줄이지 말 것)
- 각 번호 항목 아래 sub-bullet은 **1~2개**
- 전체 sub-bullet 합계는 **최대 8개**
- 각 sub-bullet은 **1~2문장, 대략 60~120자 수준**으로 충분히 설명

본문 요약 그 자체에 집중. takeaway 문장이나 "FAE 관점에서…" 같은 마무리는 **추가하지 말 것** — 영상이 말하지 않은 일반론을 덧붙이지 않습니다.

**주요 타임라인**
영상 전체 흐름을 대표하는 시간 링크 3개만 작성합니다.
- 형식: `- [[HH:MM](https://www.youtube.com/watch?v=VIDEO_ID&t=SECONDS)] 핵심 장면/주장 한 줄`
- 초반/중반/후반에서 각각 의미 있는 지점을 고릅니다.
- 긴 영상에서 초반 몇 분만 나열하지 않습니다.
- 자막 구간이 없으면 `[자막 기반 타임라인 없음]`을 1회 작성합니다.

영어 채널도 반드시 한국어로 작성합니다.

예시 골격:

```markdown
**핵심 요약**
실리콘밸리 대표 VC 마크 안드리센이 파이터 미디아와 진행한 1시간짜리 인터뷰의 요약입니다. 그는 자신이 30년간 보아온 청국 수준의 카테고리, 즉 '실행의 실수'보다 '놓친 기회의 실수'가 훨씬 치명적이라는 관점을 술명하고, 이것을 바탕으로 위대한 창업자를 식별하는 기준과 실리콘밸리 투자 모델 자체의 구조까지 확장해 나갑니다.

1. **벤처 캐피털의 사고방식: '실수로부터 배우기'의 함정**
   - 안드리센은 $10M을 잏은 손실 사례보다 Google이나 Facebook 같은 회사에 일찍 투자하지 못해 놓친 $100B 규모의 기회비용이 어떤 패턴을 가지는지를 더 고민해야 한다고 강조합니다.
   - 이 관점이 a16z의 디스쿨 문화와 투자 의사결정 구조에 그대로 반영되어 있으며, '안전한 실패'보다 '큰 승리의 누락'을 더 아프게 다뢹니다.

2. **위대한 창업자의 3가지 조건**
   - High IQ, 용기, 야망 세 가지를 모두 갖춘 사람만이 Stripe·Airbnb·OpenAI 규모의 회사를 만들 수 있다고 보며, 세 가지가 교집합처럼 겹치는 사람이 드물기 때문에 1원관 투자의 힌트율이 낮다고 설명합니다.

3. **소비자 잊여짐(Consumer Surplus)을 투자 렌즈로**
   - 안드리센은 제품이 창출하는 가치 중 기업이 가져가는 몫은 상대적으로 작고 대부분은 소비자에게 돌아간다는 점을 골드마크 지표로 제시하며, 이 구조가 클수록 필연적으로 경쟁자가 알기 어려워진다고 주장합니다.
   - 따라서 '경쟁이 없어 보이는 쿨한 시장'이 몇 년 뒤 가장 큰 기회가 되는 경우가 많다고 덧붙입니다.

**주요 타임라인**
- [[03:00](...&t=180)] 실행의 실수보다 놓친 기회비용이 더 크다는 관점
- [[14:26](...&t=866)] 위대한 창업자를 판별하는 High IQ 기준
- [[35:30](...&t=2130)] 단순 문제 해결을 넘어 시장을 재정의하려는 야망
```

## 규칙
- 영상 제목은 **반드시 h2 헤딩 + YouTube 링크** 형식 (`## [제목](https://www.youtube.com/watch?v=VIDEO_ID)`)
- 조회수, 길이, 자막 유무, **업로드 날짜**는 출력에 포함하지 않음
- 날짜 정보는 페이지 제목에 이미 포함되어 있어 영상별 날짜는 불필요
- 섹션 순서: **한 줄 인사이트** → **핵심 요약** → **주요 타임라인**
- 핵심 요약은 도입 2~3문장 + **번호 매긴 소주제(굵은 소제목)** 3~5개 + 각 소주제 아래 1~2개 sub-bullet 구조 사용 (sub-bullet 총합 최대 8개)
- sub-bullet은 1~2문장으로 각 60~120자 수준, 자막의 **구체적 인명·기업명·숫자·연도**를 포함해 설명 (모호한 일반 진술 금지)
- **주요 타임라인**은 3개 링크만 작성하고, 초반/중반/후반을 대표해야 함
- **핵심 요약 본문에 takeaway/FAE 관점/실무 적용 같은 일반론 덧붙이지 말 것** — 영상이 실제로 말한 내용만 정리
- 핵심 요약 본문에 발표자가 든 예시/사례/데모가 핵심 이해에 중요하면 **최대 2개**까지 포함 가능
- **핵심 요약 첫 문장을 영상 제목(또는 제목의 앞 부분)으로 시작하지 않음** — 제목은 이미 h2 헤딩에 있으므로 반복 금지
- **한 줄 인사이트는 모든 영상에 대해 서로 다르게 작성** — 같은 다이제스트 안에서 동일한 보일러플레이트 인사이트 금지
