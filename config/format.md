# 요약 출력 포맷

## 채널 헤더
각 채널 섹션은 h3 헤딩 + 📺 + **채널명(핸들 아님)** + **YouTube 채널 링크**:

```markdown
### 📺 [채널명](https://www.youtube.com/@CHANNEL_HANDLE)
```

예시:
- `### 📺 [AI Engineer](https://www.youtube.com/@aiDotEngineer)`
- `### 📺 [안될공학 - IT 테크 신기술](https://www.youtube.com/@unrealtech)`
- `### 📺 [데키랩](https://www.youtube.com/@dekilab)`

채널명과 핸들은 raw 데이터에서 사용. publish 단계에서 채널명이 빨간색으로 렌더링됨 (Notion `color: red`).

## 영상별 요약 포맷

각 영상 요약은 아래 형식을 정확히 따라야 합니다.

---

## [영상 제목](https://www.youtube.com/watch?v=VIDEO_ID)

**한 줄 인사이트**
💡 이 영상의 핵심 통찰을 한 문장으로. 본문에서 가장 중요한 한 가지 주장/숫자/판단을 골라 압축. (FAE 관점이나 "실무 적용" 같은 일반론을 강제하지 말 것 — 영상 자체의 내용에 충실)

**핵심 요약**
도입 1~2문장으로 영상의 전체 맥락(누구의 발언/인터뷰/데모인지, 무엇을 다루는지)을 잡아주고, 그 다음 **번호 + 굵은 소제목** 구조로 3~5개의 핵심 포인트를 정리합니다.

각 번호 항목은:
- 굵은 소제목(주제 한 줄)
- 그 아래 1~3개의 sub-bullet 또는 짧은 문단
- sub-bullet 안에는 **구체적 인명/기업명/숫자/연도**를 자막에서 그대로 가져옴
- 해당 포인트가 등장하는 시점을 **인라인 타임스탬프** `[[HH:MM](https://www.youtube.com/watch?v=VIDEO_ID&t=SECONDS)]` 형태로 본문 끝에 첨부

본문 요약 그 자체에 집중. takeaway 문장이나 "FAE 관점에서…" 같은 마무리는 **추가하지 말 것** — 영상이 말하지 않은 일반론을 덧붙이지 않습니다.

영어 채널도 반드시 한국어로 작성합니다.

예시 골격:

```markdown
**핵심 요약**
실리콘밸리 대표 VC 마크 안드레센의 인터뷰 요약. 실수의 종류, 위대한 창업자 조건 등 정리.

1. **벤처 캐피털의 사고방식: '실수로부터 배우기'의 함정**
   - 실수의 종류: '실행의 실수(Commission)' vs '방치의 실수(Omission)'. $10M 손실보다 Google 같은 기업 투자 기회를 놓치는 $100B 기회비용을 더 경계. [[03:00](...&t=180)]
   - 과거 실패 경험(Hot Stove)이 1945년 이후 AI 겨울을 지나온 투자자들의 판단을 흐리게 만듦. [[01:50](...&t=110)]

2. **위대한 창업자의 3가지 조건**
   - High IQ: 대화 중 노트를 많이 적게 만드는지로 판별. [[04:26](...&t=266)]
   - 용기(Courage): 정면 돌파 결단력. [[05:00](...&t=300)]
   - 야망(Drive): 단순 문제 해결을 넘어 자기 것을 구축하려는 욕구. [[05:30](...&t=330)]
```

**주요 타임라인** (선택 사항 — 인라인 타임스탬프가 본문에 3개 이상 있으면 생략 권장)
구조화 포맷에서는 sub-bullet마다 인라인 타임스탬프가 박혀있으므로 별도 타임라인 섹션은 불필요. 굳이 추가할 경우에만 아래 형식:
- [HH:MM:SS](https://www.youtube.com/watch?v=VIDEO_ID&t=SECONDS) 내용 1

타임라인 문구는 실제 세그먼트 내용을 요약해 작성하고, 임의 시간 생성 금지.

---

## 규칙
- 영상 제목은 **반드시 h2 헤딩 + YouTube 링크** 형식 (`## [제목](https://www.youtube.com/watch?v=VIDEO_ID)`)
- 조회수, 길이, 자막 유무, **업로드 날짜**는 출력에 포함하지 않음
- 날짜 정보는 페이지 제목에 이미 포함되어 있어 영상별 날짜는 불필요
- 섹션 순서: **한 줄 인사이트** → **핵심 요약** → (선택) **주요 타임라인**
- 핵심 요약은 도입 1~2문장 + **번호 매긴 3~5개 소주제(굵은 소제목)** + 각 소주제 아래 1~3개 sub-bullet 구조 사용
- sub-bullet에는 **구체적 인명·기업명·숫자·연도**를 자막에서 그대로 인용 (모호한 일반 진술 금지)
- 각 sub-bullet 또는 소주제 끝에 **인라인 타임스탬프** `[[HH:MM](youtube_url&t=SECONDS)]` 형태로 첨부 (본문에서 바로 점프 가능하도록)
- **핵심 요약 본문에 takeaway/FAE 관점/실무 적용 같은 일반론 덧붙이지 말 것** — 영상이 실제로 말한 내용만 정리
- 핵심 요약 본문에 발표자가 든 예시/사례/데모를 1개 이상 포함
- **핵심 요약 첫 문장을 영상 제목(또는 제목의 앞 부분)으로 시작하지 않음** — 제목은 이미 h2 헤딩에 있으므로 반복 금지
- **한 줄 인사이트는 모든 영상에 대해 서로 다르게 작성** — 같은 다이제스트 안에서 동일한 보일러플레이트 인사이트 금지
- 타임라인 timestamps는 반드시 YouTube 링크 형태로: `[HH:MM:SS](https://www.youtube.com/watch?v=VIDEO_ID&t=SECONDS)`
