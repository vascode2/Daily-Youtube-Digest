# 요약 출력 포맷

각 영상 요약은 아래 형식을 정확히 따라야 합니다.

---

### [영상 제목]

| 항목 | 내용 |
|------|------|
| 조회수 | [숫자] |
| 업로드 | [YYYY-MM-DD] |
| 길이 | [MM:SS] |
| 자막 | [있음/없음] |

**핵심 요약** (2-4문장)
> 영상의 핵심 내용을 엔지니어 관점에서 간결하게 요약합니다.

**주요 타임라인**
- [HH:MM:SS] 내용 1
- [HH:MM:SS] 내용 2
- [HH:MM:SS] 내용 3

**한 줄 인사이트**
💡 FAE 관점에서 이 영상이 업무에 주는 시사점 한 문장.
