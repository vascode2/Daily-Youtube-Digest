# 요약 출력 포맷

## 채널 헤더
각 채널 섹션은 h3 헤딩 + 📺 + **채널명(핸들 아님)**:

```markdown
### 📺 채널명
```

예시:
- `### 📺 AI Engineer` (@aiDotEngineer 핸들의 채널명)
- `### 📺 안될공학 - IT 테크 신기술`
- `### 📺 데키랩`

채널명은 raw 데이터의 `channelName` 필드를 사용. publish 단계에서 자동으로 빨간색으로 렌더링됨 (Notion `color: red`).

## 영상별 요약 포맷

각 영상 요약은 아래 형식을 정확히 따라야 합니다.

---

## [영상 제목](https://www.youtube.com/watch?v=VIDEO_ID)

**핵심 요약** (2-4문장)
> 영상의 핵심 내용을 엔지니어 관점에서 간결하게 요약합니다.

**주요 타임라인**
- [HH:MM:SS] 내용 1
- [HH:MM:SS] 내용 2
- [HH:MM:SS] 내용 3

> 자막이 없으면 "주요 타임라인" 섹션은 생략 가능합니다.

**한 줄 인사이트**
💡 FAE 관점에서 이 영상이 업무에 주는 시사점 한 문장.

---

## 규칙
- 영상 제목은 **반드시 h2 헤딩 + YouTube 링크** 형식 (`## [제목](https://www.youtube.com/watch?v=VIDEO_ID)`)
- 조회수, 길이, 자막 유무, **업로드 날짜**는 출력에 포함하지 않음
- 날짜 정보는 페이지 제목(예: "Weekly Digest 2026-04-24 ~ 2026-04-30")에 이미 포함되어 있어 영상별 날짜는 불필요
